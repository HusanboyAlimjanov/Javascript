o'zgaruvchi vireable yaratish usullari:


1.var - bu glabal o'zgaruvchi yaratish uchun ishlatiladi, var kalit so'zida yaratilgan o'garuvchilarni qayta yaratsa bo'ladi, bar bu eski o'zgaruvchi yaratish usuli.


2.let - bu blockskop o'zgaruvchi yaratish uchun kerak bo'ladi, let kalit so'zi orqali yaratilgan o'zgaruvchilar qayta yaratilib bo'lmaydi, letyangi yaratuvchi yaratish usuli.


3.const - o'garmas o'garuvchi yaratish uchun ishlatiladi, const orqali yaratilgan vireablega o'zgartirib bo'lmaydigan qiymat berilishi kerak, const yangi ozgaruvchi raratish usuli


4.alert() - cleint message(xabar) chiqarish uchun.


4.prompt() - cleintga savol berish va qiymat kiritish uchun ishlatiladi.


4.confirm() - cleintga qisqa savol berish uchun.

birinchisi - primitive

ikkinchisi - nonprimitive

primitive - 1. - string, 2. - number, 3. - bigint, 4. - boolean, 5. - undefinet, 6. - null, 7. - symbol

nonprimitive - 1. - object yozilishi - {}, 2. - array yozilishi - [], 3. - function yozilishi function funcName(){code...}

type conversion - bir data typedagi malumotni boshqa bir data typega o'zgarib ketishiga aytiladi

% - qoldiqli bo'lish

() - amalni birinchi bajarish uchun ishlatiladi

= - qiymat biriktirish uchun ishlatiladi

++ - bitta oshirib beradi

-- - bitta kamaytirib beradi

** - darajani bildiradi






Taqqoslashlar

bular conparisionlar

== , ===, >, <, >=, <=,

bulardan qaytadigan natija doim "boolean" data type kiruvchi qiymat bo'ladi

boolean: true, false

== - vazifasi: faqat qiymatlarni o'zini taqqoslaydi

==== - vazifasi: ham qiymatlarni taqqoslaydi ham data type ni ham taqqoslaydi

< - vazifasi: ikkita qiymatdan birini kichikligini tekshiradi

> - vazifasi: ikkita qiymatdan birini kattaligini tekshirib beradi

<= - vazifasi: ikkita qiymatdan biri kichikligi yoki tengligini tekshirib beradi

>= - vazifasi: ikkita qiymatdan biri kattaligini yoki tengligini tekshirib beradi

|| - (yoki operaatori) - berilgan qiymatlar ichidan trueni qidiradi agar true topilmasa eng oxirgi qiymatni chiqarib beradi

&& (va operaatori) - berilgan qiymatlar ichidan folse ni qidiradi folse topilmasa oxirgi qiymatni chiqarib beradi

! - (imkor belgisi) - berilgan qiymatni boolean data typega o'girib kegin uni teskarisiga almashtirib chiqaradi

?? -(nullish belgisi) - berilgan qiymatlar ichidan trueni qidiradi agar topilmasa oxirgisini chiqaradi "yoki (||)" operatoridan Farqi

Taqqoslashlar

🔹 % (qoldiq)
Ma’nosi: bo‘lishdan qolgan qoldiq

let a = 10 % 3; // 1

👉 Ko‘p ishlatiladi:

let son = 7;
console.log(son % 2); // 1 (toq)
🔹 ** (daraja)

Ma’nosi: darajaga oshiradi

let a = 2 ** 3; // 8
🔹 ++ (increment)

Ma’nosi: 1 ga oshiradi

let a = 5;
a++; // 6

Farqi:

let a = 5;
console.log(a++); // 5
console.log(++a); // 7
-- (decrement)

Ma’nosi: 1 ga kamaytiradi

let a = 5;
a--; // 4
+= (qo‘shib yozish)

Ma’nosi: o‘ziga qo‘shadi

let a = 5;
a += 3; // 8

Bu:

a = a + 3;
/= (bo‘lib yozish)

Ma’nosi: o‘zini bo‘ladi

let a = 10;
a /= 2; // 5
Kichik real misol:
let pul = 100;

pul += 50; // 150
pul -= 20; // 130
pul *= 2; // 260
pul /= 2; // 130
pul %= 3; // 1

console.log(pul);
Eng oson eslab qolish:
+ - * / → oddiy matematika
% → qoldiq
** → daraja
++ -- → 1 ga o‘zgartiradi
+= /= → qisqa yozish


if, else if, else

🔹 1. if — (agar)
Ma’nosi: Agar berilgan shart rost (true) bo‘lsa, kod ishlaydi.

let son = 10;

if (son > 0) {
console.log("Musbat son");
}

👉 Agar son > 0 rost bo‘lsa, ichidagi kod bajariladi.

🔹 2. else if — (aks holda agar)

Ma’nosi: Birinchi if noto‘g‘ri bo‘lsa, boshqa shartni tekshiradi.

let son = -5;

if (son > 0) {
console.log("Musbat");
} else if (son < 0) {
console.log("Manfiy");
}

👉 if ishlamasa, else if tekshiriladi.

🔹 3. else — (aks holda)

Ma’nosi: Agar yuqoridagi barcha shartlar noto‘g‘ri bo‘lsa, oxirgi variant ishlaydi.

let son = 0;

if (son > 0) {
console.log("Musbat");
} else if (son < 0) {
console.log("Manfiy");
} else {
console.log("Nol");
}

👉 Hech qaysi shart to‘g‘ri bo‘lmasa, else ishlaydi.

🔹 4. switch — (tanlash operatori)
manosi: bir o'zgaruvchini bir necha qiymat bilan solishtirib mosini bajariladi
let kun = 3
switch (kun){
case 1:
console.log("Dushanba");
break;
case 2:
console.log("Seshanba");
break;
case 3:
console.log("CHorshanba");
break;
default:
console.log("boshqa kun")
}

Qisqa farqi

if -> bitta shart tekshiradi

else if -> qo'shimcha shartlar

else -> oxirgi variant

switch -> ko'b variantni tartibli tekshiradi

case - > switch ichida tekshiriladigan aniq qiymat, yani o'zgaruvchi qaysiqiymatga ega teng bo'lsa, o'sha case ishlaydi

let kun = 2;
switch (kun) {
case 1:
console.log("Dushanba");
break;
case 2:
console.log("Seshanba");
break;
}
Bu yerda:
case 1 → agar kun = 1
case 2 → agar kun = 2


bu yerda:


case 1 -> agar kun = 1


case 2 -> agar kun = 2


break -> agar break bo'lmasa kegingi caselar ham ishlayveradi
